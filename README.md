# Movie-Search-App
This is a simple movie search app. 
You can search by title, rate movies from 1-5 stars, and add to favorites.
Requirements: install requirements.txt, aquire your API key at "https://www.omdbapi.com/apikey.aspx", assign your API key with "$env:OMDB_API_KEY = "Your API key"". 
Run the app with "uvicorn main:app --reload".
